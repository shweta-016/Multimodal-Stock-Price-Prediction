# Multimodal-Stock-Price-Prediction

This project is a Multimodal Stock Price Prediction System that combines historical stock price data, technical indicators, and financial news sentiment analysis to improve stock price forecasting accuracy. The system uses deep learning models such as LSTM and GRU for time-series forecasting and Natural Language Processing (NLP) for sentiment analysis of financial news.

The project integrates multiple data sources including stock market APIs and news APIs to collect real-time data, making the system capable of real-world stock prediction and analysis.
